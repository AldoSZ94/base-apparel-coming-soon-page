# 👗 Base Apparel Coming Soon Page

Proyecto de maquetación web basado en el desafío **"Base Apparel Coming Soon Page"** de Frontend Mentor.  
El objetivo fue construir una landing page responsive con un formulario de suscripción funcional, aplicando HTML semántico, SCSS y JavaScript moderno para validación de formularios.

---

## 📌 Descripción

Este proyecto consiste en una página de lanzamiento (*coming soon page*) para una tienda de moda, diseñada con enfoque responsive y optimización visual.

Incluye un formulario de captura de correo electrónico con validación personalizada, retroalimentación visual en tiempo real y manejo dinámico de errores mediante JavaScript.

Se puso especial énfasis en:

- Estructura semántica  
- Diseño responsive  
- Validación de formularios con JavaScript  
- Accesibilidad web  
- Código limpio y mantenible  
- Organización modular de estilos con SCSS  
- Optimización de imágenes para distintos dispositivos

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile y desktop  
- 📩 Formulario de suscripción con validación de correo electrónico  
- ⚠️ Manejo dinámico de errores mediante JavaScript  
- 🖱️ Interactividad basada en eventos (`submit` e `input`)  
- 🎨 Transiciones suaves para mejorar la experiencia visual  
- 🖼️ Implementación de imágenes responsivas con `<picture>` y formatos optimizados (`AVIF`, `WebP` y fallback)  
- 🏷️ Uso de metodología **BEM** para una estructura CSS escalable  
- ♿ Implementación de accesibilidad con `label`, `aria-label` y `autocomplete`  
- 🧩 Organización modular de estilos con **SCSS**  
- ⚡ Entorno de desarrollo moderno con **Vite**  

---

## 🧠 Aprendizaje

Durante este proyecto puse en práctica conceptos clave como:

- Manipulación del DOM con JavaScript  
- Uso de `addEventListener()` para manejar eventos del formulario  
- Validación nativa de formularios mediante `validity.valid`  
- Implementación de validación personalizada con `novalidate`  
- Separación de responsabilidades mediante funciones reutilizables  
- Uso de *early return* para mejorar legibilidad del código  
- Diseño responsive con media queries  
- Implementación de tipografía fluida mediante `clamp()`  
- Optimización de imágenes con `<picture>`, `AVIF` y `WebP`  
- Organización de estilos mediante metodología BEM  
- Uso de transiciones CSS para mejorar la experiencia de usuario  
- Desarrollo de interfaces modernas con Vite
